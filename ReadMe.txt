Quartz Composer Application

DesktopRender					An application that explains how to use a Quartz Composer 
						composition to render a XML file directly on desktop.

Sample Requirements				The project was created using the Xcode running under 
						Mac OS X 10.6.x or later. The supplied Quartz Composer 
						compositions were created using the Quartz Composer editor 
						running under Mac OS X 10.6.x or later.

About the sample				The sample uses a Quartz Composer composition that displays 
						the contents of a XML file. In the composition, the XML file 
						is parsed and its contents are displayed item by item through 
						Quartz Composer iterator patch. When the user (alt)opt-clicks 
						on the window, the editor window for the XML location is shown. 
						After the URL editing is finished, the contents of the XML 
						file will be displayed.

Using the Sample				Open the project using the Xcode editor which can be found 
						in /Developer/Applications. Build the project and run. 
						option-click on the window to edit the URL of the XML file.
